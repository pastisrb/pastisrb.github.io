Au programme : lecture de code

La lecture de code écrit par d'autres est une des pratiques les plus efficaces pour apprendre de nouveaux idiomes, acquérir de nouvelles tactiques et découvrir des motifs réutilisables.

Nous proposerons quelques courtes portions de code (moins de 100 lignes) et nous en ferons une lecture collective, analytique et critique.

Pour vous préparer, vous pouvez regarder cette présentation de Saron Yitbarek et choisir des bouts de code qui vous intriguent.

Une place pour le dotRB à gagner !

Gagnez une place gratuite pour la conférence @dotRBeu !

Pour participer, signalez une erreur ou un manque manifeste dans la doc de la StdLib (2.0 ou 2.1), en envoyant un tweet accompagné d'un petit gist @pastisrb :)

Le gagnant sera celui qui a trouvé l'erreur (ou manque) le plus grave. @davidawea et @jlecour annonceront un gagnant le soir du prochain pastis.rb !
